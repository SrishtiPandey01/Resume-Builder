# Resume Builder

A responsive **Resume Builder Web Application** that allows users to create professional resumes quickly and easily. This project is built using **HTML, CSS, JavaScript, and Bootstrap** and provides a simple interface for entering personal information, education, skills, and work experience.

The application generates a clean and structured resume layout that users can preview and print.

---

## Features

- Easy-to-use resume form
- Add personal details, education, skills, and experience
- Instant resume preview
- Clean and professional resume layout
- Fully responsive design
- Print or download resume option

---

## Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout design  
- **Bootstrap** – Responsive UI framework  
- **JavaScript** – Dynamic functionality and DOM manipulation

## Installation

Follow these steps to run the project locally.

1. Clone the repository

```
git clone https://github.com/SrishtiPandey01/resume-builder.git
```

2. Open the project folder

```
cd resume-builder
```

3. Open `index.html` in your browser.

---

## How It Works

1. User enters information in the resume form.
2. JavaScript collects the entered data.
3. The resume template is automatically generated.
4. Users can preview and print their resume.
